# 🥁 Free Alesis SR-16 Drum Samples (WAV)

A free drum sample pack from the legendary **Alesis SR-16 drum machine**, used in countless recordings since the 90s.

This pack includes carefully sampled WAV files from the real hardware, ready to use in any DAW or sampler.

---

## 🎧 What's included

* High-quality WAV drum samples
* Kicks, snares, hi-hats and percussion
* Clean, usable sounds for modern production
* Classic SR-16 character

---

## 💻 Compatibility

These samples work with:

* Any DAW (Ableton, FL Studio, Logic Pro, Reaper, etc.)
* Any sampler (Kontakt, Battery, MPC, etc.)
* Any platform (Windows / macOS)
